Getting Started

Getting started with Flying Potato is incredibly simple. Choose your preferred integration method below:

Using JSDelivr CDN

Step 1: Navigate to your HTML or JavaScript file

Step 2: Browse the src directory in the repository

Step 3: Add one of the following snippets:

For HTML

HTML 
<script src="https://cdn.jsdelivr.net/gh/flying-potato-js/v1@main/src/[file_location]"></script>

For JavaScript

JavaScript 
const utils = require('https://cdn.jsdelivr.net/gh/flying-potato-js/v1@main/src/[file_location]');
PRO TIP

For optimized performance, use the minified version by appending .min.js to your file name (e.g., utils.min.js)
